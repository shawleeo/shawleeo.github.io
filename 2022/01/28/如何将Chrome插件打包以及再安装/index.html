如何将Chrome插件打包以及再安装

Posted on 2022-01-28 Edited on 2022-01-29 In Computer , Software , Chrome

打包chrome插件

在chrome的扩展程序页签，勾选开发者模式，点击打包程序，拓展程序根目录选择要打包的程序，秘钥空着就行，再点击打包拓展程序即可。

1、密钥文件无用可删除
2、如何获取拓展程序根目录：
chrome浏览器输入

在个人资料路径里找到Extensions文件夹同时找到extension的ID方便找到相应文件夹

手动安装chrome插件

首先在 Chrome 設置 -> 更多工具 -> 擴充 (chrome://extensions/)，將右上角開發人員模式給啟用，以目前情況來說直接把 crx 文件檔案拖曳過去，會出現
CRX_HEADER_INVALID
或
crx required proof missing

將網路下載來的 CRX 外掛擴充功能，將副檔名由 .crx 改為 .zip，並且直接拖曳到擴充功能頁籤就可以順利執行運作了。
